# Move It with Next 💜

##Technologies 💡 
    - Typescript
    - React
    - Next.js


## How to run 🔧
Clone Repository
$ git clone https://github.com/RafaelGoulartB/move.it.git


## Run Web Project 📝💻
- Install Dependencies
$ yarn install

- Run Aplication
$ yarn dev

- Go to http://localhost:3000/ to see the result.
